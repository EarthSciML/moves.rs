moves.rs — WebAssembly Demo

Runs the moves.rs port of EPA MOVES entirely in the browser — no server, no install. Outputs are returned as downloadable Parquet files.

Build the WASM package first: wasm-pack build --target web crates/moves-wasm then serve this directory and open index.html. See demo/README.md for full instructions.

Onroad simulation

Upload a RunSpec XML file. The engine runs the onroad emission calculator chain and returns one or more Parquet output files matching the canonical MOVES output schema.

Select a RunSpec XML file to enable the button.
